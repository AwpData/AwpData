Walkthrough

Home

1. Type "poster" to lower the poster on the wall. This will allow you to grab the door key

2. Type "key" to grab the door key.

3. Type "magnifying glass" to grab the magnifying glass on the bookcase.

4. Type "left" to move to the computer side.

5. Type "rag" to pick up the rag off the table.

6. Type "right" twice to get to the window side.

7. Type "key" to grab the key for the power box.

8. Type "rag" to use the rag to clean the window.

9. Type "magnifying glass" to put the magnifying glass on the window so you can see the sign.

10. Type "look" to look at the sign. Notice the code is 7253, you will need this for the door

11. Type "right" to move to the to the door side

12. Type "key" to turn on the keypad on the door so you can enter the code.

13. Type "7253" to unlock the door.

14. Type "door" to open the door (you can keep typing "door" to open and close the door).

15. Type "forward" to move into the room.

16. Type "light" to turn on the light in the room so you can look around.

17. Type "right" twice to get to the door side of this room.

18. Type "door" to close the door (so you can view a poster on the left adjacent side).

19. Type "left" to get to the poster.

20. Take note of the colors on the poster in that order (red, orange, blue).

21. Type "left" twice to get to the power box.

22. Type "key" to use the key that you picked up from the window side to turn on the power box.

23. Type the code "red orange blue" to turn on the power.

24. Type "left" twice and then "forward" to return to the bedroom.

25. Type "left" to get to the computer side.

26. Type "computer" to turn in your assignment and win the game!

Secret Achievement Guide
Achievement Title How To Do It
My Bum! Type "back" when the door is closed either in the bedroom location or light bulb location
Already Memorized Type the code "7253" on the door (when possible) WITHOUT cleaning the window
Electric Feel Type "wire" in the power box location with the power on to shock yourself
I Just Felt Like Cleaning Pick up the rag and clean the window after the power is on since you do not need the code anymore
